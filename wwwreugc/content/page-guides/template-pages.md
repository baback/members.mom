# Writing Guide: Script Template Pages (`/templates/[slug]`)

## Overview

These are mid-funnel pages for people actively looking for ready-to-use video scripts for a specific niche. The reader has a product, knows they need UGC-style video, and wants a script they can use TODAY. This is the most actionable page type — the scripts ARE the content. Everything else is supporting context.

## Tone & Voice

- Write like a UGC copywriter sharing their swipe file
- Direct, practical, zero fluff — the reader wants scripts, give them scripts
- Casual and conversational in the scripts themselves (they should sound like a person talking to a camera)
- Brief and confident in the surrounding copy
- Show craft — the scripts should be genuinely good, not generic fill-in-the-blank templates

## Total Word Count

**1,500–2,200 words** per page. The scripts themselves should be ~60% of the word count. Supporting copy is minimal.

## Section-by-Section Breakdown

### 1. H1 + Meta
- Format: "UGC Video Script for [Niche]: Ready-to-Use Templates"
- Meta description: Include niche, number of scripts, and a hook
- Example: "3 ready-to-use UGC video scripts for skincare products. Testimonial, problem-solution, and hook-demo-CTA formats with 5 hook variations."

### 2. Opening Paragraph (60–80 words)
- 1 stat about why UGC works for this niche
- 1 sentence about what's on this page
- That's it. Get to the scripts fast.
- Example: "UGC-style video ads convert 2.4x better than branded content for skincare products. Below are 3 ready-to-use scripts — testimonial, problem-solution, and hook-demo-CTA — plus 5 hook variations you can swap in. Copy, customize, create."

### 3. Script Template 1: Testimonial Format (250–350 words)
- **Label**: "Script 1: The Testimonial" with a subtitle like "Best for: building trust with cold audiences"
- **Duration**: State the target length (e.g., "15–30 seconds")
- **The full script** in a visually distinct container (bordered box, shaded background):

  ```
  [HOOK — 0-3s]
  "Okay so I've been using [product] for about 3 weeks now and I have to be honest..."
  
  [SETUP — 3-8s]
  "I've tried literally everything for [problem]. [Competitor type] didn't work. 
  [Other solution] was too expensive. I was about to give up."
  
  [PAYOFF — 8-20s]
  "But [product] actually [specific result]. Like, I noticed a difference 
  within [timeframe]. [Show product / show result]"
  
  [CTA — 20-25s]
  "If you're dealing with [problem], just try it. Link in bio."
  ```

- **Below the script**, add 3 customization notes:
  - "Swap [problem] for your customer's #1 pain point"
  - "Replace [timeframe] with a realistic result window — don't overpromise"
  - "The 'I have to be honest' hook creates anticipation — keep it"
- **The script must sound like a real person talking.** Use contractions, filler words ("like", "literally", "okay so"), and natural pauses. If it reads like marketing copy, rewrite it.

### 4. Script Template 2: Problem-Solution Format (250–350 words)
- Same structure as Script 1 but different format
- **Label**: "Script 2: The Problem-Solution" with subtitle "Best for: demonstrating product benefits"
- **Duration**: 15–30 seconds
- Full script in container with stage markers
- The problem section should be emotionally resonant — the viewer should think "that's me"
- The solution section should be specific, not vague ("it cleared my skin in 2 weeks" not "it changed my life")
- 3 customization notes below

### 5. Script Template 3: Hook-Demo-CTA Format (250–350 words)
- Same structure
- **Label**: "Script 3: The Hook-Demo-CTA" with subtitle "Best for: product-focused ads, showing the product in action"
- **Duration**: 15–20 seconds (shorter — this format is punchy)
- This format is more visual — the script should include [SHOW] directions:
  ```
  [HOOK — 0-3s]
  "Stop scrolling if you [have problem / want result]"
  
  [DEMO — 3-12s]
  [Show product being used / applied / opened]
  "Watch this — [describe what's happening on screen]"
  
  [CTA — 12-15s]
  "Get yours before they sell out. Link below."
  ```
- 3 customization notes below

### 6. 5 Hook Variations (150–200 words)
- 5 alternative hooks that can be swapped into ANY of the 3 scripts above
- Each hook:
  - The line in quotes
  - What trigger it uses in parentheses: (curiosity), (social proof), (controversy), (urgency), (relatability)
  - 1 sentence on when to use it
- Examples for skincare:
  1. *"I can't believe I'm showing you my skin without makeup right now..."* (vulnerability) — Great for before/after reveals
  2. *"My dermatologist is going to hate me for saying this..."* (controversy) — Creates intrigue, high thumb-stop rate
  3. *"POV: you finally found a product that actually works"* (relatability) — Native TikTok format, feels organic
  4. *"$12 and it outperforms my $80 serum"* (value comparison) — Price-conscious audiences
  5. *"3 weeks ago my skin looked completely different"* (transformation) — Sets up a before/after story
- **These hooks should be niche-specific. Generic hooks = generic page.**

### 7. Tips for Customizing These Scripts (100–150 words)
- 4–5 quick tips in bullet format:
  - "Replace product details but keep the emotional structure intact"
  - "Match the actor's demographic to your target audience"
  - "Test each script with 3 different hooks before judging performance"
  - "Shorter is almost always better for cold traffic. Cut anything that doesn't earn its seconds."
  - "Add captions — 85% of social video is watched on mute"
- Keep this section brief. The scripts are the star.

### 8. CTA Section
- Standard CTA component
- Customize: "Turn these scripts into videos in minutes. No camera needed."
- Button: "Get Started"

## Formatting Rules

- Scripts MUST be in visually distinct containers — not inline text. Use bordered boxes, shaded backgrounds, or code-block styling
- Stage markers ([HOOK], [SETUP], [PAYOFF], [CTA]) should be bold and clearly visible
- Duration labels should be prominent near each script title
- Hook variations should be numbered and in blockquote/italic styling
- Link to: matching industry page, related showcase page, relevant create page
- Add "Niche: [niche]" and "Scripts: 3" as metadata near the H1

## What NOT to Do

- Don't write scripts that sound like blog posts read aloud. They should sound like someone talking to their phone camera
- Don't use formal language in scripts ("I would like to share my experience with..."). Use casual language ("Okay so I've been using this for like 3 weeks...")
- Don't make all 3 scripts sound the same. Each format should feel distinctly different
- Don't skip the customization notes — they're what make templates actually usable
- Don't write generic hooks that work for any niche. If you can swap "skincare" for "supplements" and the hook still works, it's too generic
- Don't pad the page with long intros or explanations. Scripts first, context second
